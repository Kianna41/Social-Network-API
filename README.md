# Social-Network-API

## Expected Outcome:

1. When the command is entered into the terminal, the server is started and the Mongoose models are synced to the MongoDB database.
2. When the user open API GET routes in Insomnia for users and thoughts, the data for each of these routes is displayed in a formatted JSON.
3.  When the user test API POST, PUT, and DELETE routes in Insomnia, they are able to successfully create, update, and delete users in the database.
4. When the user test API POST and DELETE routes in Insomnia, they are able to successfully create and delete user's friends in the database.




Link to GitHub Repository: https://github.com/Kianna41/Social-Network-API
